# Política de Segurança

Este repositório contém o código-fonte do meu portfólio pessoal e site profissional. Levo a segurança a sério e aprecio a ajuda da comunidade para manter este projeto seguro.

## Versões Suportadas

Como este é um projeto pessoal de implantação contínua (Continuous Deployment) via GitHub Pages, apenas a versão mais recente (branch `master`) recebe atualizações de segurança e manutenção.

| Versão | Suportada          |
| -------| ------------------ |
| Latest | :white_check_mark: |
| Antigas| :x:                |

## Reportando uma Vulnerabilidade

Se você descobrir alguma vulnerabilidade de segurança neste projeto (como dependências expostas, falhas no Next.js ou XSS), por favor, **não crie uma issue pública**. 

Em vez disso, siga estes passos:

1. Envie um e-mail diretamente para **ewerton.v50@gmail.com** com o assunto `[Vulnerabilidade de Segurança] - Portfólio`.
2. Inclua uma descrição clara do problema e, se possível, os passos para reproduzir a vulnerabilidade.

### O que esperar após o reporte:
* Você receberá uma confirmação de recebimento do seu e-mail em até 48 horas.
* Avaliarei o problema e, se confirmado, trabalharei em uma correção (patch) o mais rápido possível.
* O alerta do Dependabot e as atualizações do `yarn.lock` são monitorados semanalmente.

Agradeço antecipadamente por reportar problemas de forma responsável!
